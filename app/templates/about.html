Acerca de SOC Agent

Plataforma de threat intelligence para analistas SOC

Qué es SOC Agent

SOC Agent es una plataforma diseñada para acelerar la investigación de indicadores de compromiso (IOCs) — IPs, dominios, hashes y URLs. Combina múltiples fuentes de threat intelligence, búsqueda web OSINT y modelos de lenguaje (LLMs) para producir veredictos accionables, con contexto histórico y correlación entre análisis.

19 APIs de Threat Intel

Consulta en paralelo VirusTotal, AbuseIPDB, Shodan, GreyNoise, OTX, ThreatFox, URLhaus, MalwareBazaar, SecurityTrails, Hybrid Analysis, Criminal IP, Pulsedive, URLScan, Google Safe Browsing, Shodan InternetDB y más, según el tipo de IOC.

Búsqueda Web OSINT

Un agente de dos pasos usa Tavily para buscar contexto adicional en la web cuando las APIs estructuradas no son suficientes: un LLM planifica las búsquedas y otro sintetiza los resultados con citas por URL.

5 LLMs con Routing

Un orquestador selecciona entre distintos proveedores de LLM (xAI Grok, OpenAI GPT-4, Groq LLaMA, Google Gemini y más) para generar resúmenes ejecutivos, hallazgos clave y recomendaciones accionables.

Memoria entre Sesiones

El chat interactivo recuerda análisis previos de un IOC entre sesiones, permitiendo consultas de seguimiento con contexto histórico completo.

Grafo de Correlación

Detecta y visualiza relaciones entre IOCs analizados previamente, ayudando a identificar campañas e infraestructura compartida entre amenazas.

MITRE ATT&CK

Mapeo automático de técnicas MITRE ATT&CK detectadas a partir de los datos de threat intelligence, con soporte para deep analysis por IOC.
